# Netflix Movie Data Analysis

## Project Description
This project analyzes Netflix movie data using Python.  
The analysis helps to understand movie trends, genres, release years, and ratings available on Netflix.

The project is implemented in a Jupyter Notebook and focuses on data cleaning, exploration, and visualization.

2. Open the project folder
3. Start Jupyter Notebook
4. Open the file `movie data analysis netflix.ipynb`
5. Run all cells to see the analysis
